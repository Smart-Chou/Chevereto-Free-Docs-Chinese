# Chevereto

Chevereto-Free 允许您在自己的服务器上创建功能齐全的图像托管网站。这是您的托管和规则，告别关闭和限制。

# ✨ 屏幕

# 🦓 关于这个分支

Chevereto-Free 是 Chevereto V3.16.2在新窗口打开 的一个分支，其中只保留了基本功能。 Chevereto-Free 供个人使用和小型社区使用。

👉 这个分支去掉的

  • 社交网络登录
  • 外部存储服务器
  • 喜欢和追随者
  • 管理横幅
  • 多语言支持
  • 适度
  • 批量内容导入

👉 这个叉子缺少的

  • V3.17在新窗口打开, V3.18在新窗口打开 和 V3.20在新窗口打开 中添加的所有更新(非安全)包括：
    • 360 图像
    • 嵌套专辑
    • ImageMagick 支持
    • 改进的共享选项
    • 用户界面升级
    • 12FA 支持
    • 命令行接口

# 🧐 要求

  • PHP 7.4
  • MySQL 5.7 / 8 - MariaDB 10
  • Apache HTTP Web Server
    • mod_rewrite
