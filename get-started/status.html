# 版本

# 当前版本

3.20.12在新窗口打开 coqueto - Chevereto V3.20 LTS(长期支持)。

此版本正在进行积极维护和更新，直到 2022-11 用于：

  • 安全更新
  • PHP 更新
  • 软件错误
  • 弃用的依赖项

Chevereto V3.20 LTS 是使用 Ubuntu 20.04在新窗口打开 制作的，因此您应该能够在 2025-04 之前运行基于 V3.20 的系统。

版本LTS 结束生命终结
V3.20.X2022-112025-04

# 下一个版本

V3.20 是 3.X 的最终版本。不会有下一个版本，只是维护修订版 (3.20.X)。
